# 📄 AI Document Similarity Checker

An AI-powered document similarity checker built with Python and SpaCy. The application compares two text documents using Natural Language Processing (NLP) techniques and provides a semantic similarity score along with a plagiarism assessment.

This project was developed to explore text preprocessing, semantic analysis, and document comparison using modern NLP libraries.

# ✨ Features
Compare two text inputs
Semantic similarity analysis using SpaCy
Automatic text preprocessing
Stopword removal
Lemmatization
Lowercase normalization
Plagiarism percentage calculation
Low, Medium, and High plagiarism detection
Input validation for empty text
Clean and modular Python implementation
Beginner-friendly code structure

# 🛠 Technologies Used
Python 3
SpaCy
Natural Language Processing (NLP)

# 📖 How It Works
The user enters two text passages.
The application preprocesses both texts by:
Converting text to lowercase
Removing stopwords
Removing punctuation
Applying lemmatization
SpaCy generates vector representations of the processed text.
The semantic similarity score is calculated.
The application displays:
Similarity score
Similarity percentage
Plagiarism level
